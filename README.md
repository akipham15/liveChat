# live-chat

## Project setup
```
npm install
```

## Project setup firebase config
```
/src/firebase/config.js must update firebaseConfig
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### run for production
```
npm install -g serve
serve -s dist
```
